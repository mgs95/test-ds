# Button

## Regular button component

Use button to perform all actions related with user interaction. Buttons should also be used to perform routing actions.

<Preview withToolbar>
  <Story name="standard">
    <Button label='Hello Button' onClick={action('clicked')} primary />
  </Story>
</Preview>

<Props of={Button} />

## Usage

Button is used to perform actions that required human interaction. By default the button shows an some text and no action. It is possible to redirect users to other pages by using tht `to` property. Making a link like this will transform the button into a `a` HTML element.


### Styles

4 styles are supported.
